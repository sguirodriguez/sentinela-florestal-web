-------Operadores-------
+ -> soma
- -> subtração
* -> multiplicação
/ -> divisão

= -> atribuição de valor
== -> comparação de valor
!= -> diferente 
|| -> OU, algebra boolean, pode ser 1 ou 2
&  -> valor tem quer ser maior que 10 e menor que 15
> -> valor maior que
< -> valor menor que
>= -> valor maior ou igual que
<= -> valor menor ou igual que
! -> validação se nao existe 

------Variaveis----
const -> constante (um valor que não muda)
var   -> variável global, consegue acessar fora do escopo
let   -> variável que voce consegueu acessar apenas no escopo e voce consegue alterar


const VALOR_DA_CARNE = 25,00

HTML -> linguagem de marcação
<input/> --> caixas de texto, posso inserir nome ou data
<button/> ---> pressionar o botao, ao pressionar faz algo (cadastro, login)
<p></p>   --> escrever algo na tela (margens, tamanho diferente)
<h2></h2> --> titulo  h1,h2,h3,h4
<span></span> --> escreve algo na tela
<div></div> ---> Pensar como uma caixa na tela, estiliza, defini tamanho
 

 CSS -> ESTILIZA nosso HTML
 width -> largura
 height -> altura
 padding -> espacamento, retrai a propria caixa
 margin -> espacamento, empurra as outras caixas
 background-color -> cor da caixa
 color -> cor de texto
 display -> flex, column
